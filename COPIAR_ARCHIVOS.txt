PROYECTO PYTHON CON INTERFAZ PROFESIONAL
========================================

1. Descomprime el ZIP.
2. Abre la carpeta sistema_rutas_puno en VS Code.
3. Ejecuta la interfaz gráfica:

   python run.py

   o también:

   python run_gui.py

4. Para usar la versión de consola:

   python run_cli.py

5. Para probar el sistema:

   python -m unittest discover -s tests -v
